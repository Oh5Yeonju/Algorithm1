
## 🔎_02. 나머지가 1이 되는 수 찾기
<b>문제 설명</b>
</br>
> 자연수 n이 매개변수로 주어집니다. n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 return 하도록 solution 함수를 완성해주세요. 답이 항상 존재함은 증명될 수 있습니다.

> </br>

<b>제한 조건</b>
>- 3 ≤ n ≤ 1,000,000


><b>
입출력 예</b>
| num |	result |
|--|--|
| 10	| 3 |
| 12	| 11 |

<br>

### ✍️ _풀이_

```js
function solution(n) {
    for(let i = 2; i <= n; i++){
        if(n % i === 1) return i; 
    }
}
```


### ✨느낀점 
오랜만에 하니 for 형식도 헷갈림 ㅜ 작은 자연수를 찾아야하므로 찾자마자 바로 i값을 리턴하는걸로 끝냄
